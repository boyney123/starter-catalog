## Overview

The `OrderAmended` event is published when an existing order is modified. As of version 0.0.3, the event always includes comprehensive details on amendments, and amendments are recorded consistently following transactional logic in the Orders Service.

```json title="Example Payload"
{
  "orderId": "123e4567-e89b-12d3-a456-426614174000",
  "userId": "123e4567-e89b-12d3-a456-426614174000",

    {
```
