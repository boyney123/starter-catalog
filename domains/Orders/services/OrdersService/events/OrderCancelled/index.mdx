## Overview

The `OrderCancelled` event is triggered whenever an existing order is cancelled. The event payload now always includes a `cancellationReason` field, which is kept up-to-date as part of improved workflows and validation. This ensures all relevant services can respond accurately, whether due to customer request, inventory issues, or validation failures.
