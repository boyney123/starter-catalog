## Overview

The `OrderRefunded` event is triggered whenever an existing order is refunded, either fully or partially. This event ensures that downstream systems such as Finance and Notification services are informed about refund activity to initiate appropriate actions (e.g., issuing payments, sending customer notifications) without coupling directly to cancellation flows.
