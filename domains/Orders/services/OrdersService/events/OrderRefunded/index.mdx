## Overview

The OrderRefunded event is triggered when a processed order has been refunded back to the customer, either fully or partially. This event ensures that all relevant services are notified of refunds, so downstream consumers (e.g., accounting, analytics, customer-facing status updates) can react accordingly and keep the system consistent.
